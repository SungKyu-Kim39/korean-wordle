# 📝 오늘의 단어 (Korean Wordle)

> 5번의 기회 안에 숨겨진 한글 단어를 맞추는 웹 기반 퍼즐 게임입니다. 
> 세계적으로 유행한 '워들(Wordle)'을 한글 자모 조합 방식에 맞게 재해석했습니다.

👉 **[게임 바로 플레이하기](https://sungkyu-kim39.github.io/korean-wordle/)** *(<- 이 링크는 설정하신 GitHub Pages 주소로 확인해 주세요)*

## 🎮 게임 방법

1. 주어지는 기회는 **총 5번**입니다.
2. 화면의 가상 키보드를 이용해 **자음과 모음 총 5개**로 이루어진 단어를 입력합니다. 
   - *예시: '가방' 입력 시 👉 `ㄱ` + `ㅏ` + `ㅂ` + `ㅏ` + `ㅇ`*
3. 5글자를 모두 채운 후 **[제출하기]** 버튼을 누르면, 타일의 색상이 변하며 힌트를 제공합니다.
   - 🟩 **초록색:** 자음/모음이 정답에 포함되어 있고, **위치도 정확히** 맞습니다.
   - 🟨 **주황색:** 자음/모음이 정답에 포함되어 있지만, **위치가 다릅니다**.
   - ⬛ **회색:** 정답에 **포함되지 않은** 자음/모음입니다.
4. 타일의 색상을 힌트 삼아 5번 안에 정답 단어를 유추해 보세요!

## ✨ 주요 기능

* **직관적인 UI와 애니메이션:** 원작의 감성을 살린 부드러운 타일 플립 및 팝 애니메이션.
* **한글 맞춤형 가상 키보드:** 표준 두벌식 배열을 차용하여 모바일과 PC 환경 모두에서 터치/클릭으로 쉽게 입력 가능. (ㅐ, ㅔ 등 이중모음 지원)
* **물리 키보드 연동:** PC 환경에서는 실제 키보드를 타이핑하여 플레이할 수 있도록 단축키(Enter, Backspace 등) 이벤트 지원.
* **정적 파일 서빙:** 별도의 백엔드 서버 없이 HTML, CSS, JS와 JSON 파일만으로 완벽하게 구동 가능.

## 🛠 기술 스택

* **Frontend:** HTML5, CSS3, Vanilla JavaScript
* **Database:** JSON (`words.json`을 통한 단어장 관리)
* **Hosting:** GitHub Pages
